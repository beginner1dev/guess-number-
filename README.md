#Number Guessing Game  | بازی حدس عدد توسط کار

##Description
This is a simple Python project where the computer picks a number between 1 and 99 and the user then has to guess the correct number.
Along the way, the computer will prompt the user whether the guessed number is greater than the number they chose (telling them to guess a smaller number) or
smaller (telling them to guess a larger number) until the user gets to the correct number | این یک پروژه ساده با پایتون است که کامیپوتر یک عدد بین 1 تا 99 انتخاب میکند و سپس کاربر باید با حدس زدن به عدد درست برسد .
در طی مسیر کامیپوتر به او راهنمایی میکند که عدد حدسی کاربر بزرگتر از عدد انتخابی خودش است (پیام می‌دهد که باید عدد کوچکتری حدس بزند) 
یا کوچکتر است (پیام می‌دهد که باید عدد بزرگتری حدس بزند) تا کاربر به عدد درست برسد .

##Technologies
-Python3
##Features
-Random number generation | تولید عدد تصادفی
-Receive input from the user | دریافت ورودی از کاربر
-Guidance in every guess | راهنمایی در هر حدس 
-End with a correct guess and ask the question of whether to play again | پایان با حدس درست و پرسش از کاربر که دوباره بازی میکند یا خبر

##How to run | نحوه اجرا
1.Install Python3 | نصب پایتون3 
2.Save the code as guess.py | ذخیره کردن کد با نام guess.py
3.Run | اجرا
